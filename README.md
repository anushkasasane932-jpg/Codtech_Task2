# Flip-Flop & Counter Build

## Project Title
D Flip-Flop and 4-Bit Counter Using Verilog HDL

## Objective
To design and simulate a D Flip-Flop and a 4-bit binary counter using Verilog HDL.

## Components Implemented
- D Flip-Flop
- 4-Bit Binary Counter

## Software Used
- EDA Playground
- Icarus Verilog

## Files Included
- designf.v.rtf
- designc.v.rtf
- testbenchf.v.rtf
- testbenchc.v.rtf
- flipflop_output.png
- flipflop_waveform.png
- counteroutput.png
- counterwaveform.png

## Result
The D Flip-Flop stored data correctly on the rising edge of the clock, and the 4-bit counter counted from 0000 to 1111 successfully.
